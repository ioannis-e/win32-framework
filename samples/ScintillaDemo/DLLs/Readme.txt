About these Scintilla dll files
-------------------------------

The Scintilla dll files provided here were extracted from files downloaded 
from https://www.scintilla.org/SciTEDownload.html.

The 32bit Scintilla dll file was extracted from https://www.scintilla.org/wscite32_553.zip
The 64bit Scintilla dll file was extracted from https://www.scintilla.org/wscite553.zip

The version of these files is 5.5.3.

32 bit programs requires the 32 bit scintilla.dll, and 64 bit programs require
the 64 bit scintilla.dll.
